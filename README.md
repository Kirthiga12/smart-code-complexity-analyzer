# Smart Code Complexity Analyzer

A static code analysis tool that evaluates Python code complexity using AST parsing.

## Features

- Function detection
- Loop detection
- Conditional detection
- Nesting depth calculation
- Complexity score generation
- Optimization suggestions

## Tech Stack

### Backend
- FastAPI
- Python AST
